## Mismo modelo — diferentes entornos

Usa la CLI de Aspire para levantar todo localmente o crear artefactos de despliegue en CI/CD. [Construye tu primera app con Aspire](/es/get-started/first-app/).

<CardGrid>
  <a href="/es/reference/cli/commands/aspire-run/" class="clickable-card">
    <Card title="Desarrollo local" icon="laptop">

    <Code code={'aspire run'} lang='bash' frame='none' />

    - Se ejecuta con runtime local de contenedores
    - Dashboard disponible en `localhost`
    - Usa secretos, volúmenes y config de desarrollo
    </Card>
  </a>

  <a href="/es/reference/cli/commands/aspire-deploy/" class="clickable-card">

    - Misma estructura, despliega donde quieras
    - Servicios cloud gestionados o infraestructura propia
    - Se integra sin fricción con pipelines CI/CD
    </Card>
  </a>
</CardGrid>
