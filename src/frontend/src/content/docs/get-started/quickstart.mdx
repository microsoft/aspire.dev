## How to use Codespaces samples

Once you launch a Codespace, follow these steps:

<Steps>

1. **Wait for setup** (1-2 minutes)
   
   The devcontainer installs the Aspire CLI and dependencies automatically.

2. **Start the CodeTour**
   
   Look for the **CodeTour** panel in the VS Code sidebar (Explorer tab). Click **"Start Tour"** to begin the guided walkthrough.
   
   <Aside type="tip">
     CodeTour shows you around the code with explanations at each step. Use the **Next** and **Previous** buttons to navigate.
   </Aside>

3. **Run the application**
   
   Open the integrated terminal (<kbd>Ctrl</kbd>+<kbd>\`</kbd>) and run:
   
   <Code code="aspire run" lang="bash" title="Terminal" />
   
   The Aspire Dashboard opens automatically in a new tab.

4. **Explore the dashboard**
   
   The dashboard shows:
   - **Resources:** All running services and containers
   - **Console logs:** Real-time output from services
   - **Structured logs:** Filtered, searchable logs
   - **Traces:** Distributed tracing across services
   - **Metrics:** Performance data and health checks

5. **Make changes**
   
   Edit the code, and hot reload will update your running app automatically. Try:
   - Modifying the API response
   - Changing the frontend UI
   - Adding a new service

</Steps>
