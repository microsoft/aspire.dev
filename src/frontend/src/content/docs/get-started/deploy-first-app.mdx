Deploy your first Aspire app by choosing the AppHost language you used for the orchestration layer. The deployment walkthrough follows that same AppHost, so this choice lets you stay in a familiar toolchain while deciding whether your guide uses `AppHost.cs` / `apphost.cs` or `apphost.ts`.

<Aside type="note" title="Start with the AppHost language">
Choose the AppHost language that matches the toolchain you want to use for the orchestration layer. Either AppHost can deploy apps in multiple languages.
</Aside>
