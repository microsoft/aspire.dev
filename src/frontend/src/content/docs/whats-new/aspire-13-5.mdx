## 🔒 Security updates

### Visual Studio Code extension dependency update

The Aspire Visual Studio Code extension's npm dependencies have been updated to address a security vulnerability.

**CVE-2026-46681** — a Prototype Pollution vulnerability in the `@nevware21/ts-utils` package (via `objDeepCopy`/`objCopyProps` using `for...in` without `hasOwnProperty` guards) — has been resolved by upgrading to a patched version of the library. The vulnerability also affected `setValueByKey` and `setValueByIter`.

Update the Aspire extension in Visual Studio Code to receive this fix.
