## Upgrade to Aspire 13.5

<span id="upgrade-to-aspire-13-5"></span>
<br />

For general upgrade guidance, see [Upgrade Aspire](/whats-new/upgrade-aspire/).

The standard upgrade flow is to update the Aspire CLI first, then update your projects from the root of your repository:

<Steps>

1. Update the Aspire CLI itself:

   ```bash title="Aspire CLI - Update the CLI"
   aspire update --self
   ```

1. Update your projects:

   ```bash title="Aspire CLI - Update all Aspire packages"
   aspire update
   ```

</Steps>

Or install the CLI from scratch:

<OsAwareTabs syncKey="terminal">
  <Fragment slot="unix">

    ```bash title="Aspire CLI - Install Aspire CLI"
    curl -sSL https://aspire.dev/install.sh | bash
    ```

  </Fragment>
  <Fragment slot="windows">

    ```powershell title="Aspire CLI - Install Aspire CLI"
    irm https://aspire.dev/install.ps1 | iex
    ```

  </Fragment>
</OsAwareTabs>

<LearnMore>
  For more details on installing the Aspire CLI, see [Install the
  CLI](/get-started/install-cli/).
</LearnMore>
