---
title: Azure App Service Client integration
description: Learn about Azure App Service client integration.
---

Azure App Service is a hosting integration only. There is no separate client integration package for Azure App Service itself.

When you add an Azure App Service environment in your AppHost, supported compute resources are deployed to Azure App Service. Your projects keep using the same client integrations they already use for databases, caches, message queues, and other dependencies.

The Azure App Service integration changes the hosting target and deployment flow. It doesn't add a new runtime client library to your application projects.

For more information on configuring resources to run on Azure App Service, see [Azure App Service Hosting integration](/integrations/cloud/azure/azure-app-service/azure-app-service-host/).
