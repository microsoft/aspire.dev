<Aside type="caution" title="Deprecated Package">

As of Aspire 13, the functionality from **CommunityToolkit.Aspire.Hosting.Python.Extensions** has been incorporated into the official [Aspire.Hosting.Python](/integrations/frameworks/python/python-host) package. This Community Toolkit package is now deprecated.

**Please use the official [Python Hosting integration](/integrations/frameworks/python/python-host) for all new development.**

</Aside>

The Aspire Community Toolkit Python hosting extensions package was a community-driven effort that provided Python hosting capabilities before official support was available in Aspire. With Aspire 13, Python became a first-class workload with all these features now built into the official package.

## Migration Guide

If you're currently using `CommunityToolkit.Aspire.Hosting.Python.Extensions`, follow these steps to migrate:

### 1. Update Package References

Remove the Community Toolkit package and install the official package:

```xml title="AppHost.csproj"
<!-- Remove this -->
<PackageReference Include="CommunityToolkit.Aspire.Hosting.Python.Extensions" Version="*" />

<!-- Add this -->
<PackageReference Include="Aspire.Hosting.Python" Version="13.0.0" />
```

<InstallPackage packageName="Aspire.Hosting.Python" />

### 2. Update Using Statements

Replace Community Toolkit imports with official package imports:

```csharp
// Remove this
using CommunityToolkit.Aspire.Hosting.Python.Extensions;

// Add this (if not already present)
using Aspire.Hosting;
```

### 3. Update Method Calls

The API is largely the same, but resource types have changed:

**AddUvicornApp:**
```csharp
// Before (Community Toolkit)
var python = builder.AddUvicornApp("python-api", "../python-app", "main:app");

// After (Official - same API)
var python = builder.AddUvicornApp("python-api", "../python-app", "main:app");
```

**AddUvApp:**
```csharp
// Before (Community Toolkit)  
var python = builder.AddUvApp("python-api", "../python-app", "main.py");

// After (Official - use WithUv() method)
var python = builder.AddUvicornApp("python-api", "../python-app", "main:app")
    .WithUv();
```

### Benefits of Migration

The official package provides:

- ✅ **Better Integration**: Full support in Visual Studio and VS Code
- ✅ **Automatic Virtual Environments**: Smarter venv detection and management  
- ✅ **More Features**: Additional methods like `AddPythonApp`, `AddPythonModule`, `AddPythonExecutable`
- ✅ **Active Maintenance**: Regular updates and bug fixes from the Aspire team
- ✅ **Unified Experience**: Consistent API with other Aspire hosting packages
- ✅ **Debugging Support**: Zero-config debugging in VS Code
- ✅ **Automatic Dockerization**: Production-ready Dockerfile generation
