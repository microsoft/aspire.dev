## 始めよう

Aspire を使い始めるには、次の手順をご利用ください:

<Steps>

1. [事前準備をチェック](/ja/get-started/prerequisites/): 必要なツールと SDK がインストールされていることを確認してください。
1. [Aspire CLI をインストール](/ja/get-started/install-cli/): 開発マシンに Aspire のツール群をセットアップしてください。
1. [最初のアプリを作成する](/ja/get-started/first-app/): クイックスタート ガイドに従って、最初の Aspire アプリケーションを作成してください。
1. [最初のアプリをデプロイする](/ja/get-started/deploy-first-app/): Aspire のリソースをデプロイする方法を学んでください。

</Steps>

## 主要な概念を確認しよう

自信を持って分散アプリケーションを構築する準備はできていますか？
Aspire は開発ワークフローを効率化し、複雑なマルチサービス アーキテクチャを簡単に作成・デプロイ・デバッグできるようにします。

    — 複雑さを排除する統合ツール群が、その秘密です。
    <LinkButton
