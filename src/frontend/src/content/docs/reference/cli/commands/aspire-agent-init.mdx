## Name

`aspire agent init` - Set up the Aspire skill file and related configuration for detected AI coding agent environments.

## Synopsis

```bash title="Aspire CLI"
aspire agent init [options]
```

<AsciinemaPlayer src="/casts/agent-init.cast" poster="npt:0:02" rows={15} loop={false} />

## Description

The `aspire agent init` command prepares your project for AI coding agents. It detects supported agent environments (such as VS Code with GitHub Copilot, Claude Code, Copilot CLI, and OpenCode) and writes the files each one expects.

The command performs the following:

- Detects supported agent environments in the current workspace
- Installs the **Aspire skill file** (recommended) so the agent knows how to drive Aspire via the CLI
- Optionally installs the **Playwright CLI** skill for browser automation
- Optionally configures the **Aspire MCP server** (legacy — not recommended alongside the skill file)

For guidance on which components to select, see [Use AI coding agents](/get-started/ai-coding-agents/).
