## Name

`aspire mcp` - Manage MCP (Model Context Protocol) server.

## Synopsis

```bash title="Aspire CLI"
aspire mcp [command] [options]
```

## Description

The `aspire mcp` command provides subcommands for managing the Aspire MCP (Model Context Protocol) server. The MCP server integrates Aspire into your AI development ecosystem, allowing AI assistants to interact with your Aspire resources, telemetry, and integrations.

## Options

- <Include relativePath="reference/cli/includes/option-help.md" />

## Subcommands

| Command                                      | Description                                                      |
| -------------------------------------------- | ---------------------------------------------------------------- |
| [`aspire mcp init`](../aspire-mcp-init/)     | Initialize MCP server configuration for detected agent environments. |
| [`aspire mcp start`](../aspire-mcp-start/)   | Start the MCP (Model Context Protocol) server.                  |

## Examples

- Initialize MCP configuration for your AI assistant:

  ```bash title="Aspire CLI"
  aspire mcp init
  ```

- Start the MCP server:

  ```bash title="Aspire CLI"
  aspire mcp start
  ```
