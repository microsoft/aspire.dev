## Quick Start

All dependencies have been installed automatically (`pnpm i` was run during setup).

**To start the development server:**

```bash
pnpm dev
```

## Useful Tips

- Open the terminal with <kbd>Ctrl</kbd>+<kbd>`</kbd> (or <kbd>Cmd</kbd>+<kbd>`</kbd> on Mac)
- The dev server will hot-reload as you make changes
- Open the site to review changes at: [http://localhost:4321](http://localhost:4321) (this is port forwarded automatically by Codespaces)
